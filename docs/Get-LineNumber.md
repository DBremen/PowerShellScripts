# Get-LineNumber

## SYNOPSIS
Retrieve specific Linenumber(s) from afile.

## Script file
Data Wrangling\Get-LineNumber.ps1

## SYNTAX

```
Get-LineNumber -Path <Object> [-LineNumber] <Object>
```

## DESCRIPTION
Retrieve specific Linenumber(s) from a file.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
dir .\test.txt | Get-LineNumber 1,3,4
```
