Ftp: remotehelp

Applies To: Windows 7,Windows Vista,Windows Server 2008,Windows Server 2008 R2

Displays help for remote commands. For examples of how this command can be used, see Examples.

Syntax

 Copy Code
remotehelp [<Command>]

Parameters

Parameter Description

[<Command>]

Specifies the name of the command about which you want help. If Command is not specified, ftp displays a list of all remote commands.

Remarks

You can run remote commands using quote or literal.

Examples

Display a list of remote commands.

 Copy Code
remotehelp

Display the syntax for the feat remote command.

 Copy Code
remotehelp feat
