Dirquota

Applies To: Windows Server 2008

Adds, lists, modifies and deletes quotas.

Syntax

 Copy Code
Dirquota {Quota | Autoquota | Template | Admin | Notification}

Parameters

Quota

List, add, modify, and delete quotas.

Autoquota

List, add, modify, and delete auto apply quotas.

Template

List, add, modify, and delete quota templates.

Admin

Configure settings and perform administrative operations.

Notification

Display help on notification configuration file syntax.
