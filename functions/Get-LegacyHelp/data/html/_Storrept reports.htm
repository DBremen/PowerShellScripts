Storrept reports

Applies To: Windows Server 2008 R2,Windows Server 2008

Lists, schedules, modifies, deletes, and cancels storage reports tasks. This command also generates storage reports on demand.

Note
To use this command, you must be logged on as an administrator or a member of the Administrators group.

Syntax

 Copy Code
storrept reports list
storrept reports add
storrept reports modify
storrept reports delete
storrept reports generate
storrept reports cancel

Parameters

Parameter Description

Storrept reports list

Lists currently configured storage report tasks.

Storrept reports add

Schedules a new storage report task.

Storrept reports modify

Modifies an existing storage report task.

Storrept reports delete

Deletes a storage report task.

Storrept reports generate

Generates a storage report on demand.

Storrept reports cancel

Cancels currently running or queued storage report tasks.
