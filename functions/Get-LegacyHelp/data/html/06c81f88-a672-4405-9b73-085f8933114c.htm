Services for Network File System Command Reference

Applies To: Windows Server 2008,Windows Vista

Services for Network File System (NFS) provides a file sharing solution that enables you to transfer files between computers running Windows Server 2008 and UNIX operating systems using the NFS protocol.

The following is a list of NFS command-line tools.

Command Description

mapadmin

Manage User Name Mapping for Microsoft Services for Network File System.

Mount

Mount Network File System (NFS) network shares.

Nfsadmin

Manage Server for NFS and Client for NFS.

Nfsshare

Control Network File System (NFS) shares.

Nfsstat

Display or reset counts of calls made to Server for NFS.

Rpcinfo

List programs on remote computers.

Showmount

Display mounted directories.

Umount

Remove Network File System (NFS)–mounted drives.
