Storrept

Applies To: Windows Server 2008 R2,Windows Server 2008

The storrept command is installed with File Server Resource Manager and includes subcommands for creating and managing storage reports and storage report tasks, as well as for configuring general administrative options for File Server Resource Manager.

Note
To use this command, you must be logged on as an administrator or a member of the Administrators group.

Syntax

 Copy Code
storrept reports
storrept admin

Subcommands

Subcommand Description

Storrept reports

Lists, schedules, modifies, and cancels storage reports tasks. This parameter also generates storage reports on demand.

Storrept admin

Configures default administrative options and storage report parameters.
