List

Applies To: Windows Server 2008

Lists writers, shadow copies, or currently registered shadow copy providers that are on the system. If used without parameters, list displays help at the command prompt.

For examples of how to use this command, see Examples.

Syntax

 Copy Code
list writers [metadata | detailed | status]
list shadows {all | set <SetID> | id <ShadowID>}
list providers

Parameters

Parameter Description

writers

Lists writers. See List writers for syntax and parameters.

shadows

Lists persistent and existing non-persistent shadow copies. See List shadows for syntax and parameters.

providers

Lists currently registered shadow copy providers. See List providers for syntax and parameters.

Examples

To list all shadow copies, type:

 Copy Code
list shadows all
