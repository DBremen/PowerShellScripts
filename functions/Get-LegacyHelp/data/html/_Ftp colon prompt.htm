Ftp: prompt

Applies To: Windows 7,Windows Vista,Windows Server 2008,Windows Server 2008 R2

Toggles between prompt mode on and off. For examples of how this command can be used, see Examples.

Syntax

 Copy Code
prompt

Parameters

none

Remarks

  • By default, prompt is on.

  • FTP prompts during multiple file transfers to allow you to selectively retrieve or store files. Mget and mput transfer all files if prompt is off.

Examples

Toggle prompt mode on and off.

 Copy Code
prompt
