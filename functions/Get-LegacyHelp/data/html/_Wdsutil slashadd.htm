/add

Applies To: Windows Server 2008,Windows Server 2008 R2

Subcommands

Subcommand Description

/add-Device

Prestages a computer in Active Directory.

/add-Image

Adds boot or installation images.

/add-ImageGroup

Adds an image group.

/add-DriverGroupPackage

Adds a driver package to a driver group.

/add-DriverGroupPackages

Adds driver packages to a driver group.

/add-DriverPackage

Adds a driver package to the server.

/add-ImageDriverPackage

Adds a driver package that is in the driver store to an existing boot image on the server. The image version must be Windows 7 or Windows Server 2008 R2.

/add-ImageDriverPackages

Adds driver packages from the driver store to a boot image on the server. The image version must be Windows 7 or Windows Server 2008 R2.

/add-AllDriverPackages

Adds driver packages from a folder to a server.

/add-DriverGroup

Adds a driver group to a server.

/Add-DriverGroupFilter

Adds a filter to a driver group on a server.
