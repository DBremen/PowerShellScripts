Filescrn

Applies To: Windows Server 2008

Create and manage file screens, file screening exceptions, and file groups.

Syntax

 Copy Code
Filescrn {Filegroup | Screen | Exception | Template | Admin | Notification}

Parameters

Parameter Description

Filegroup

List, add, modify, and delete file groups.

Screen

List, add, modify, and delete file screens.

Exception

List, add, modify, and delete file screen exceptions.

Template

List, add, modify, and delete file screen templates.

Admin

Configure settings and perform administrative operations.

Notification

Display help on notification configuration file syntax
