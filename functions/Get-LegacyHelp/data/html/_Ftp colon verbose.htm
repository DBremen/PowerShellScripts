Ftp: verbose

Applies To: Windows 7,Windows Vista,Windows Server 2008,Windows Server 2008 R2

Toggles verbose mode. For examples of how this command can be used, see Examples.

Syntax

 Copy Code
verbose

Parameters

none

Remarks

By default, verbose is on. When verbose is on, all ftp responses are displayed. When a file transfer is completed, statistics regarding the efficiency of the transfer are also displayed.

Examples

Toggle verbose mode on and off.

 Copy Code
verbose
